CRUD METHODS

Note that for an API the urls on many of these CRUD methods would need to be changed to the standard RESTful API syntax, including: GET /users, GET /users/:id, POST /users, PUT /users, DELETE users/:id . This demo is only illustrating what these look like on the backend.

  • /users (get all users)
  • /usersPopulate (get all users using populate)
  • /createUser (OrphanAnnie)
  • /getUser (OrphanAnnie)
  • /updateUser (Change OrphanAnnie to age 5)
  • /updateUserCar (Make Annie an owner of a Ford)
  • /deleteUser (Delete Annie)
